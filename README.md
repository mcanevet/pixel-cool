# Pixel Cool 🕹️

Un jeu de puzzle pixel art où vous devez reproduire une mosaïque en échangeant des pixels.

## Comment jouer

1. Regardez l'image "Objectif" en haut à gauche
2. Cliquez sur un pixel dans "Ton puzzle" pour le sélectionner
3. Cliquez sur un autre pixel pour échanger leurs couleurs
4. Répétez jusqu'à reproduire exactement l'objectif !

## Niveaux de difficulté

| Niveau | Grille | Pixels |
|--------|--------|--------|
| Easy | 4×4 | 16 |
| Medium | 8×8 | 64 |
| Hard | 12×12 | 144 |
| Expert | 16×16 | 256 |
| Master | 20×20 | 400 |

## Fonctionnalités

- 🏆 5 niveaux de difficulté
- 🎨 Émojis pixellisés comme images à reproduire
- 📱 Design responsive (mobile et desktop)
- 🎮 Mode détente (pas de timer, pas de limite de coups)
